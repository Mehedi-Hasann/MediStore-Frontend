# MediStore

**MediStore** is a full-stack medicine e-commerce platform that allows users to browse, order, and manage medicines online. The platform features a modern responsive frontend and a secure backend with RESTful APIs.

---

## Project Structure

---

## Features

- User registration and authentication
- Browse and search medicines
- Add medicines to cart and place orders
- Admin dashboard for managing products and orders
- Responsive and user-friendly interface
- Secure database operations with Prisma ORM

---

## Technologies Used

- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** Node.js, Express.js, Prisma ORM
- **Database:** PostgreSQL
- **Others:** JWT, Postman
