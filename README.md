# MediStore

**Backend Repo:** [https://github.com/Mehedi-Hasann/MediStore-Backend](https://github.com/Mehedi-Hasann/MediStore-Backend)  
**Live Demo:** [https://medi-store-frontend-ten.vercel.app/](https://medi-store-frontend-ten.vercel.app/)

**MediStore** is a full-stack medicine e-commerce platform that allows users to browse, order, and manage medicines online. It features a modern responsive frontend and a secure backend with RESTful APIs.

---

## Online Medicine Store Roles

There are three roles: **Admin**, **Seller**, and **Customer**.

### Admin
- Full access to all functionalities
- Can manage products, orders, users

### Seller
- Can add or remove medicines
- Track orders and manage users

### Customer
- Can register and login
- Add/remove medicines to/from cart
- Give reviews, create addresses, and modify profile

## Features

- User registration and authentication
- Browse and search medicines
- Add medicines to cart and place orders
- Admin dashboard for managing products and orders
- Responsive and user-friendly interface
- Secure database operations with Prisma ORM

## Technologies Used

- **Frontend:** Next.js, React, Tailwind CSS  
- **Backend:** Node.js, Express.js, Prisma ORM  
- **Database:** PostgreSQL  
- **Others:** JWT, Postman
